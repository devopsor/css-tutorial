Single-value syntax: A single value sets the color of all four borders.

Two-value syntax: The first value sets the color of the top and bottom borders, and the second value sets the color of the right and left borders.

Three-value syntax: The first value sets the color of the top border, the second value sets the color of the left and right borders, and the third value sets the color of the bottom border.

Four-value syntax: Each value sets the color of the border individually in the order top, right, bottom, and left.

Note: You must declare the border-style property before the border-color property. The element must have a border before the border's color can be set.
